### react 脚手架  create-react-app
1. 全局安装 mac 加 sudo
    - `sudo npm install -g create-react-app`
2. 创建项目文件
    - `create-react-app my-app`

3. 进入项目文件夹
    - `cd my-app/`
4. 启动项目
    - `npm start`
5. public文件
    - 最终的文件，会放到线上

6. index.js: 入口文件
7. App.js：组件

### 将demo放到git上
1. 初始化目录
    - `git init .`

2. 添加好文件
    - `git add .`
    -  `git status` : 查看文件内容

3. 形成版本
    - `git commit -m "react-start"`
4. 推送到git仓库里
    - `git remote add origin https://github.com/rabbit-tian/react-start.git
git push -u origin master`
